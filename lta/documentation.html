Purpose

Language Tag Analyzer (LTA) is an XSLT 2.0 implementation that checks wellformedness, validity and some other constraints of languags tags, following BCP 47.

Downloads

  • Version 0.1 (May 2008). NOTE: this version is based on a previous draft of ltr-4646-bis, so its usage is discouraged.
  • Version 0.2 (October 2008). Encompasses:
    • ANT Build script for updating XML version of language subtag registry (see also the XML Schema document for validation)
    • Main XSLT file which imports the other ones
    • localized messages for HTML output, currently in English and in German
  • Version 0.3 (February 2009). More constraints being checked and a different HTML output.
  • Version 0.4 (February 2009). A bit different HTML output.
  • Version 0.5 (July 2009). More, more output messages!
  • After version 05, moving to GitHub
